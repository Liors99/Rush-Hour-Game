General:
	1.In order to run the game, please make sure that the Car.py file is in the same folder as the version of the game you want to play.

Text version:
	1. The program will ask you what car you want to move and how many units.
	2. If you want to move the car down or to the right, enter a positive amount of units to be moved, if you want to move the car to the left, enter a negative amount of units (each dot on the grid is 1 unit).

GUI version:
	1.You can move any car by drag and dropping it to the desired position.
	2.Once a car hits a boundry or another car, you will have to click and drag it in order to move it again.
	3.Every time you left click on a car, the total number of moves will increase by 1.
